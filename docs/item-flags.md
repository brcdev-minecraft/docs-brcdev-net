# Item Flags

We automatically support new Item Flags as they are added to the game. So if you can't find what you are looking for below, check here:
https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/inventory/ItemFlag.html

| Item flag name            | Description                                                         | Minecraft version |
|---------------------------|---------------------------------------------------------------------|-------------------|
| HIDE_ADDITIONAL_TOOLTIP   | Setting to show/hide potion effects, book/firework info, maps, etc. | 1.20.5 or newer   |
| HIDE_ARMOR_TRIM           | Setting to show/hide armor trim patterns and materials              | 1.20.5 or newer   |
| HIDE_ATTRIBUTES           | Setting to show/hide Attributes like Damage                         | 1.8 or newer      |
| HIDE_BANNER_PATTERNS      | Setting to show/hide patterns of banners                            | 1.20.5 or newer   |
| HIDE_BASE_COLOR           | Setting to show/hide the base color of banners                      | 1.20.5 or newer   |
| HIDE_BEES                 | Setting to show/hide bees inside a beehive or bee nest              | 1.20.5 or newer   |
| HIDE_BLOCK_ENTITY_DATA    | Setting to show/hide block entity data                              | 1.20.5 or newer   |
| HIDE_BLOCK_STATE          | Setting to show/hide block state information                        | 1.20.5 or newer   |
| HIDE_BREAK_SOUND          | Setting to show/hide the break sound of an item                     | 1.20.5 or newer   |
| HIDE_BUNDLE_CONTENTS      | Setting to show/hide the contents of a bundle                       | 1.20.5 or newer   |
| HIDE_CAN_BREAK            | Setting to show/hide what blocks the item can break                 | 1.20.5 or newer   |
| HIDE_CAN_PLACE_ON         | Setting to show/hide what blocks the item can be placed on          | 1.20.5 or newer   |
| HIDE_CONTAINER            | Setting to show/hide items inside a container                       | 1.20.5 or newer   |
| HIDE_CUSTOM_DATA          | Setting to show/hide custom NBT/Component data                      | 1.20.5 or newer   |
| HIDE_CUSTOM_MODEL_DATA    | Setting to show/hide the Custom Model Data value                    | 1.20.5 or newer   |
| HIDE_DESTROYS             | Setting to show/hide what the ItemStack can break/destroy           | 1.8 or newer      |
| HIDE_DYE                  | Setting to show/hide dye information                                | 1.16 or newer     |
| HIDE_DYED_COLOR           | Setting to show/hide the "Dyed" tooltip on items                    | 1.20.5 or newer   |
| HIDE_ENCHANTS             | Setting to show/hide enchants                                       | 1.8 or newer      |
| HIDE_FIREWORKS            | Setting to show/hide firework flight and explosion info             | 1.20.5 or newer   |
| HIDE_INSTRUMENT           | Setting to show/hide the instrument type (Goat Horns)               | 1.20.5 or newer   |
| HIDE_JUKEBOX_PLAYABLE     | Setting to show/hide the "Jukebox Playable" tooltip                 | 1.20.5 or newer   |
| HIDE_LODESTONE_TRACKER    | Setting to show/hide the lodestone position info on compasses       | 1.20.5 or newer   |
| HIDE_LORE                 | Setting to show/hide the item lore                                  | 1.20.5 or newer   |
| HIDE_PLACED_ON            | Setting to show/hide where this ItemStack can be build/placed on    | 1.8 or newer      |
| HIDE_POT_DECORATIONS      | Setting to show/hide decorations on a decorated pot                 | 1.20.5 or newer   |
| HIDE_POTION_CONTENTS      | Setting to show/hide specific potion effect information             | 1.20.5 or newer   |
| HIDE_POTION_EFFECTS       | Setting to show/hide potion effects on this ItemStack               | 1.8 or newer      |
| HIDE_RARITY               | Setting to show/hide the item rarity (Common, Epic, etc.)           | 1.20.5 or newer   |
| HIDE_STORED_ENCHANTMENTS  | Setting to show/hide enchantments on Enchanted Books                | 1.20.5 or newer   |
| HIDE_UNBREAKABLE          | Setting to show/hide the unbreakable State                          | 1.8 or newer      |
| HIDE_WRITTEN_BOOK_CONTENT | Setting to show/hide the content of a written book                  | 1.20.5 or newer   |
